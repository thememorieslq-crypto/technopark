**Как запустить проект**
В проекте есть два готовых файла для быстрого запуска:

- **Способ 1. Через Python**
Дважды кликните на файл `start_python.bat`. Откроется браузер с проектом, а в консоли будет работать сервер. Чтобы остановить сервер, закройте окно консоли или нажмите Ctrl+C.
- **Способ 2. Через Node.js**
Дважды кликните на файл `start.bat`. Результат будет тот же — браузер откроется автоматически.

Если ни Python, ни Node.js не установлены, можно использовать любой другой статический сервер. Например, встроенный в Python модуль (если Python всё же есть):
1. python -m http.server 8000
2. Или расширение Live Server в редакторе VS Code.
Важно: проект нужно обязательно запускать через сервер, а не просто открывать index.html в браузере. Иначе понарамы и 3D-модели не загрузятся из-за политики безопасности браузера.

**Где что лежит**
Все настройки комнат и меток находятся в файле `js/data.js`. Там можно менять названия, пути к панорамам и моделям, а также положение меток. Сами панорамы (формат jpg) лежат в папке `assets/panoramas`, а 3D-модели (формат glb) — в `assets/models`.
